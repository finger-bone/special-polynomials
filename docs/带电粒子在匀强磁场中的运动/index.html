摘自原子物理与量子力学第二版上册（科学出版社），170-172页。

如果一个量子在均匀磁场中运动，那么如果使用矢势，

A=(−By,0,0)A = (-By, 0, 0)A=(−By,0,0)

[H,px]=[H,pz]=[px,pz]=0[H,p_x] = [H,p_z] = [p_x,p_z] = 0[H,px​]=[H,pz​]=[px​,pz​]=0

我们比较关心能量，因此 HHH 是测量值的一部分。又因为哈密顿量要求确定的 yyy ，因此 pyp_ypy​ 不能选，因此用 H,px,pzH,p_x,p_zH,px​,pz​ 。
