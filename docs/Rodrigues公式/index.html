Rodrigues 公式

复习​

在物理中解方程时，我们一般只关注，

  1. 解可以被多项式表示时，本征值的要求
  2. 解的具体内容

如果要求本征解是可积函数，那么它一定可以被展开为多项式。而多项式空间总可以由幂函数张成。理论上可以将幂函数通过施密特正交化得到一组新的基作为本征解，且第 nnn 个基地没有高于 nnn 次的项。再利用完备性，展开 x∣n⟩x|n\ranglex∣n⟩ ，通过对比幂次得到递推公式。

不过其实但是并没有人要求我们只能展开 x∣n⟩x|n\ranglex∣n⟩ ，我们完全可以展开，

ddx∣n+1⟩\frac{\mathrm{d}}{\mathrm{d}x}|n+1\rangledxd​∣n+1⟩

因为它也是多项式，而且首一规定下，这个多项式最高次系数是 nnn ，一定的一样可以找到一个递推式。积分积回去之后再待定一个常数就可以了。这也算是另一个推导方法了。

不论如何，我们一定能得到一个递推公式和本征值的可能取值。这样我们理论上完整地解出来了微分方程——但是递推公式总觉得不太好。

下面我们会给出一种更好的方法，即基于厄米算子的方法直接推出 Rodrigues 公式。

我们先写一下公式，看一下我们要推的结论。

如果我们有，

yn=1w(ddx)n(hnw)y_n = \frac{1}{w} (\frac{\mathrm{d}}{\mathrm{d}x})^n(h^n w)yn​=w1​(dxd​)n(hnw)

Rodrigues 形式​

我们把，

hD2y+(hw)′wDy=λyh D^2y + \frac{(hw)'}{w} Dy = \lambda yhD2y+w(hw)′​Dy=λy

暂且叫做 Rodrigues 形式 （我自己瞎起的名）。这个形式的优势是右侧没有权函数，但是也正是因为右侧没有权函数，定义一个带权的内积空间就显得很突兀了。

前面我们的形式都是，

py′′+qy′+ry=λwypy'' + qy' + ry = \lambda w ypy′′+qy′+ry=λwy

Rodrigues 形式除了次数要求外，唯一要求是 r=0r=0r=0 。

同时，注意，带权内积空间下，厄米条件保持不变，因为，

pD2y+qDy=λwyp D^2y + q Dy = \lambda w ypD2y+qDy=λwy

的厄米条件是，

qw=(pw)′qw = (pw)'qw=(pw)′

Rodrigues 公式​

现在我们有方程，

hD2y+(hw)′wDy=λyh D^2y + \frac{(hw)'}{w} Dy = \lambda yhD2y+w(hw)′​Dy=λy

这里最重要的观察是，如果我们对上面的公式两侧同时求导，如果将 y′y'y′ 作为求解函数，右侧依然是本征值的形式。但左侧算子会变化。

hD2y+(hw)′wDy=λyh D^2y + \frac{(hw)'}{w} Dy = \lambda yhD2y+w(hw)′​Dy=λy

两侧同时求导为，

hD2y′+(Dh+(hw)′w)Dy′+D((hw)′w)y′=λy′h D^2 y' + (Dh + \frac{(hw)'}{w}) Dy' + D(\frac{(hw)'}{w}) y' = \lambda y'hD2y′+(Dh+w(hw)′​)Dy′+D(w(hw)′​)y′=λy′

此时如果有，

(hw)′w\frac{(hw)'}{w}w(hw)′​

是一次的，或者说，

D((hw)′w)=μD(\frac{(hw)'}{w}) = \muD(w(hw)′​)=μ

那么方程的形式在求导后是不变的，即，

hD2y′+(Dh+(hw)′w)Dy′=(λ−μ)y′h D^2 y' + (Dh + \frac{(hw)'}{w}) Dy' = (\lambda - \mu) y'hD2y′+(Dh+w(hw)′​)Dy′=(λ−μ)y′

同时注意，

Dh+(hw)′w=(hw)′+h′ww=(h2w)′hwDh + \frac{(hw)'}{w} \\ = \frac{(hw)' + h'w}{w} \\ = \frac{(h^2w)'}{hw}Dh+w(hw)′​=w(hw)′+h′w​=hw(h2w)′​

因此最后方程是，

hD2y′+(h2w)′hwDy′=(λ−μ)y′h D^2 y' + \frac{(h^2w)'}{hw} Dy' = (\lambda - \mu) y'hD2y′+hw(h2w)′​Dy′=(λ−μ)y′

这相当于对于方程，

hD2y+(hw)′wDy=λyh D^2y + \frac{(hw)'}{w} Dy = \lambda yhD2y+w(hw)′​Dy=λy

做了如下的代换，

h2=h1w2=h1w1λ2=λ1−μ1y→y′h_2 = h_1 \\ w_2 = h_1w_1 \\ \lambda_2 = \lambda_1 - \mu_1 \\ y \rightarrow y'h2​=h1​w2​=h1​w1​λ2​=λ1​−μ1​y→y′

μ2=D(Dh+(hw)′w)=D2h+μ1\mu_2 = D(Dh + \frac{(hw)'}{w}) = D^2h + \mu_1μ2​=D(Dh+w(hw)′​)=D2h+μ1​

因此如果方程形式要保持不变，即一阶的特征多项式保持一次，那么二阶的特征多项式必须是不高于二次的。

我们记，

η=D2h\eta = D^2 hη=D2h

因此再加一条代换规则，

μ2=μ1+η\mu_2 = \mu_1 + \etaμ2​=μ1​+η

此外注意到，上面的方程可以写成一个更方便的形式，即，

λwy=D(hwy′)\lambda w y = D(hwy')λwy=D(hwy′)

那么如果给定 y′y'y′ ，我们是可以很简单地得到 yyy 的。

我们知道方程的收敛解一定是多项式，因此我们总可以执行上面的代换 NNN 次（即多项式的次数），最后得到一个 yyy 的 nNnNnN 次导，这是的解是常数，然后将解逐次迭代上去。
